Poskytuje triedy potrebne na pracu so Spritmi a Sprite tabulu.

Package poskytuje dve triedy: Sprite a SpriteSheet.
- Sprite a jej instancia zdruzuje obrazok jednej Sprite.
- SpriteSheet je trieda, ktora v sebe obsahuje tabulkovy obrazok z ktoreho mozme pomocou metod ziskat jednotlive Sprite.
