Poskytuje triedy potrebne na vytvorenie roznych vnutornych menu

Package poskytuje triedy : InventoryMenu/Journal/CharInfo/ConversationPanel/CraftinMenu/....
Tieto triedy dediace od AbstractInMenu vytvaraju vnutorne menu pre rozne typy operacii :
- InventoryMenu = Inventar hraca
- Journal = Vypis aktivnych questov
- CharInfo = Vypis info o hracovi
- ConversationPanel = Konverzacny panel
- ...
Pomocou plugin systemu sa daju pridat dalsie a nastavovat z listenerov.
