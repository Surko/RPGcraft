Poskytuje triedy potrebne na vytvorenie/spracovanie/aktualizovanie castic.

Package poskytuje zakladnu triedu: Particles .
- Particles je trieda ktora vytvara interface pre ostatne castice :
   -> BarParticle je trieda, ktorej instancia sluzi ako zobrazovatel nejakeho progress baru podla urcitej farby
   -> TextParticle je trieda, ktorej instancia zobrazuje textovu casticu pri hracovi
