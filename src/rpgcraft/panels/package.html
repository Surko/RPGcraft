Poskytuje triedy potrebne na vytvorenie roznych druhov menu, v ktorych sa nachadzajuce komponenty.

Package poskytuje triedy ktore popisuju druhy menu
- AboutMenu je trieda pre menu s popiskom hry.
- IntroMenu je trieda pre menu s privitanim do hry.
- GameMenu je trieda pre menu s hlavnou hrou.
- LoadCreateMenu je trieda pre menu na nacitanie/vytvorenie hry.
- BlankMenu je trieda pre prazdne menu.
- FactoryMenu je trieda pre tovarenske menu (vytvorene z hry/listenerov)
- MainMenu je trieda pre hlavne menu
