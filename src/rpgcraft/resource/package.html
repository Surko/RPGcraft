Poskytuje triedy potrebne na vytvorenie a zdruzenie roznych typov resourcov/dat nacitanych z xml suborov. 
 Package poskytuje zakladne triedy : AbstractResource, StatResource a StringResource
- AbstractResource je interface pre vsetky ostatne Resource (okrem Stat a StringResource) triedy :
  -> Ui/Tile/ConversationGroup/Conversation/Quests/Image/Entity/Sound/RecipeResource su triedy, ktore v sebe obsahuju nacitane resource
    z xml suborov.
- StatResource je trieda ktora zdruzuje info o roznych statoch/atributoch entit.
- StringResource je trieda ktora ma v sebe nacitane vsetky texty ziskane z lokalizacneho suboru stringXXX.properties
