Poskytuje triedy potrebne na vytvorenie typov pre rozne druhy komponent 
 Package poskytuje zakladnu abstraktnu triedu: AbstractType. 

- AbstractType je interface pre vsetky ostatne typy, ktory zdruzuje aspon zakladne, spolocne vlastnosti/metody vsetkych typov :
  -> List/Panel/Button/Text/ImageType su rozne typy pre rozne komponenty, ktore mozme mat v hre.
